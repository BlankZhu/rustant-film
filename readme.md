## Add up information

`rustant-film` supports these following EXIF and other possible detail parameters to be shown up:

- Artist
- Lens
- Camera
- Camera manufacturer
- Aperture
- Focal length
- Shutter speed / Exposure time
- ISO
- Timestamp
- Watermark
- Invisible Watermark
